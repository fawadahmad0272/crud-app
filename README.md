# crud-app
A simple CRUD (Create, Read, Update, Delete) application built using C# and ASP.NET MVC in Visual Studio 2022. This project demonstrates basic database operations using Entity Framework and SQL Server, allowing users to manage student records efficiently.

# Features:
Add, edit, view, and delete student information
Validation using Data Annotations
Clean and responsive UI
Built with MVC architecture
Connected with SQL Server database

# Tech Stack:
C# (.NET Framework / .NET Core)
ASP.NET MVC
Entity Framework
SQL Server
Visual Studio 2022
